Preprocessing

In terms of data preprocessing, we found that performing a few more minor cleaning procedures made our descriptive statistics and analyses easier to generate. More specifically, we converted the datatype of the attribute ‘artist_followers’ to an integer from a string so that we could more easily gather descriptive statistics on these values. We also changed the datatype of the attribute ‘album_release’ from string to datetime. This enabled us to process the attribute in a more efficient manner using tools in the pandas library, including ordering our dataframe by date.
